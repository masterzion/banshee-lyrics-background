# banshee-lyrics-background

Get current song in Banshee and run songtext program in a terminal on desktop background

Install:
``` bash
sudo pip install songtext
sudo apt-get install wmctrl
``` 

Open uxterm in the first screen;

Run:
``` bash
./run.sh &
```
